# Block Blaster Pro

Classic arcade-style **Breakout/Block Breaker game** built with **HTML, CSS, and JavaScript**.  
Play directly in your browser — works on desktop and mobile! 🎮

---

## How to Play

- **Move paddle:**  
  - Desktop: `Arrow Left` / `Arrow Right`  
  - Mobile: Drag your finger across the screen
- **Break all blocks** to clear the level
- **Collect power-ups**:  
  - **Expand** → Paddle gets wider for 10 seconds  
  - **Slow** → Ball slows down for 10 seconds
- Start with **3 lives**. Lose all lives → Game Over  
- Each level increases difficulty (more blocks, faster ball, smaller paddle)  

---

## Features

- Multiple levels with increasing difficulty  
- Power-ups to help you survive longer  
- Score, Level, and Lives displayed  
- Mobile-friendly touch controls  
- Colorful block patterns
